Entendamos HTML

¿Qué es HTML exactamente? HTML proporciona estructura al contenido que aparece en un sitio web, como imágenes, texto o vídeos. Haz click derecho en cualquier página de internet, selecciona “Inspreccionar”, y verás HTML en un panel en tu pantalla. HTML significa HyperText Markup Language (Lenguaje de Marcado de HiperTexto):

  • Un lenguaje de marcado es un lenguaje de computadora que define la estructura y presentación de texto plano.
  • En HTML, la computadora puede interpretar texto plano que se encuentra rodeado por elementos HTML.
  • El HiperTexto es mostrado en una computadora o dispositivo que provee acceso a otro texto a través de enlaces, también conocidos como hiperenlaces. Probablemente hiciste click en unos cuantos hiperenlaces para llegar a este curso.
